## ⚠️ Common Network Issues & Fix Approach

### 1. VPN Not Connecting
- Cause: Credential / port / service issue
- Fix: Reset + service restart + port check

### 2. Slow Internet via Zscaler
- Cause: Policy / DNS / routing
- Fix: Reconnect client + flush DNS

### 3. Application Blocked
- Cause: Firewall deny rule
- Fix: Check logs → allow rule
